"# Recordoc_BE" 

### 작업 중 규칙

- 매 STEP마다 전체 pytest 실행 금지
- 관련 모듈 테스트만 실행
- 전체 테스트는 STEP 완료 시 또는 PR 전 실행

### 함수 생성 규칙

- 새로운 함수를 생성하면 연관된 함수의 동작 흐름을 주석으로 작성해준다.
- 함수 내에서 다른 함수를 호출할 때 넘버링으로 순서를 표기해준다.
- 함수의 필요성을 주석으로 달아준다.
---

### Plan 규칙

- 코드 구현 시 서비스 코드에 대해서는 무엇에 관한 기능인지 간략하게 주석을 달아준다
- 주석의 내용은 기능 요약, 기능 흐름, 파라미터 의미 및 예시가 들어간다.
